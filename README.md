# カプセルトイ予算管理アプリ

## 概要
カプセルトイなど「毎回決まった金額の支出」を管理することに特化したシンプルな予算管理アプリです。  
カプセルトイ用途を想定していますが、日々の固定支出などさまざまな用途にも利用できます。

## デモ
https://chamty.github.io/budget-app/

## 使い方
1. 設定画面 > 予算設定で今月の予算を設定
2. カプセルトイを回したら、ホーム画面で該当の金額ボタンをタップして支出を記録
3. 該当の金額ボタンがない場合は、設定画面 > 金額ボタン設定 で追加・編集・削除が可能

## 機能
- 固定金額の支出を簡単に記録
- 合計支出をひと目で確認
- 予算を設定
- 固定金額のカスタマイズ

## 使用技術
- Flutter / Dart
- shared_preferences（ローカルデータ保存）

## 工夫した点
固定金額の支出を素早く記録できるよう、入力の手間を最小限に抑えたUI設計にしました。
